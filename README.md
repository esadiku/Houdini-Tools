# Houdini-Tools
Houdini Shelf Tools
Houdini Tools is a collection of shelf tools for Houdini:
Houdini-Tools/
├─ shelf/
│   ├─ objMerge.shelf
│   ├─ CreateNull.shelf
│   ├─ ConnectWires.shelf
│   └─ SOPtoOBJ.shelf
├─ installer.py
├─ README.md

## Installation

git clone https://github.com/esadiku/Houdini-Tools.git
cd Houdini-Tools
python installer.py
